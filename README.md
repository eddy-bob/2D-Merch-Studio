## Getting Started

First, install dependencies:

```bash
pnpm install
# or
npm install
# or
yarn install
```

Then, run the development server:

```bash
pnpm dev
# or
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Next.js 16** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS v4** - Styling
- **merch-design-enhancer** - AI-powered design enhancement

## Usage

1. Upload your 2D design file (drag & drop or click to browse)
2. Adjust design controls (scale, rotation, opacity, color)
3. Select your merchandise type
4. Choose an AI provider and enter your API key
5. Click "Enhance with AI" to generate your merchandise mockup
6. Download the enhanced design
